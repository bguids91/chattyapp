Chatty App
=====================

A simple real-time chat web application that allows multiple users to chat with one another. 

### Dependencies

**Client Side** 

- babel-core
- babel-loader
-babel-preset-es2015
- babel-preset-react
- css-loader
- node-sass
- sass-loader
- sockjs-client
- style-loader
- webpack
- webpack-dev-server
- react
- react-dom
- boostrap

**Server Side**

- express
- ws
- uuid
